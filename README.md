## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Firebase account

### Installation

```bash
# Clone the repository
git clone https://github.com/metalheadcode/utopia-assessment.git

# Navigate to project directory
cd utopia-assessment

# Install dependencies
npm install
# or

# Set up environment variables
```
